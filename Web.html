Projet web

Le projet web consiste à mettre au format web un rapport d'annalyse environemental

Résumer

Nous devions fournir une fiche personna et un site web composé d'une page d'accueil qui donne le contexte du site, la présentation de l'entreprise, dans notre cas nous avions choisi Cdiscount, son analyse PESTE(L) et son macro-environement mais nous devions aussi créer une partie contacts avec un formulaire et rajouter les mentions légales

Rendu final

Nous avons pu créer le site responsive avec du HTML, CSS et en utilisant des flexboxs. Nous avions également créé une fiche persona qui nous avait permis d'orienter le style graphique du site sur quelque chose de simple et épurer.
